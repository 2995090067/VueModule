，把本地仓库和远程仓库链接一起    从本地上传至GitHub空间

// 在项目文件夹内执行以下命令
	1.git init //初始化一个本地仓库
	
	2.git add ./ //将所有文件添加至本地仓库的门口
	
	3.git commit -m "上传备注信息"
	
	4.git remote add origin 你的空间git链接
	
	5.git push -u origin master
